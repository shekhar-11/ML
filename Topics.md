Core Machine Learning Algorithms

1. Regression (predict numbers)
Linear Regression (you know this)
Polynomial Regression
Ridge & Lasso Regression

Example: Predict house price

2. Classification (predict categories)
Logistic Regression
Decision Trees
Random Forest
K-Nearest Neighbors (KNN)
Naive Bayes
Support Vector Machine (SVM)

Example: Spam or Not Spam

3. Clustering (unsupervised learning)
K-Means Clustering
Hierarchical Clustering
DBSCAN

Example: Group customers based on behavior

4. Neural Networks / Deep Learning
Artificial Neural Networks (ANN)
Convolutional Neural Networks (CNN) (for images)
Recurrent Neural Networks (RNN) (for sequences)
Transformers (modern AI like ChatGPT)

Example: Image recognition, speech, NLP

5. Dimensionality Reduction
PCA (Principal Component Analysis)
t-SNE

Example: Reduce features for visualization

6. Reinforcement Learning
Q-Learning
Deep Q Networks (DQN)

Example: Game playing, robotics
